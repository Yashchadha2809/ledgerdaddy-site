Security Overview

This is an informational summary. Review with counsel and your security team before publishing.

Last updated:

Controls

  • Encryption in transit (HTTPS/TLS); encryption at rest where supported by our providers.
  • Least-privilege access and role-based permissions.
  • Audit logs for key actions and access events.
  • Secure software development lifecycle and vulnerability management.

Data handling

Production data is restricted to authorized staff. Backups are performed on a scheduled basis. Data is processed only to deliver the service.

Incident response

We investigate incidents, remediate, and notify customers as required by law and contract.

Compliance

We follow industry practices for security and privacy. Formal audit statements, if any, will be listed here.
